🏮 モツオベーダー (Motsuo-Vader)

「一献、お待たせしました。」

下町の聖水「梅割り」を操り、襲来するモツ（敵）を捌きまくる、居酒屋シューティングゲーム。
🍶 ゲーム概要

迫りくるモツ（文字）をショットで撃ち落とし、店を回し続けるアーケードスタイルのゲームです。
特定のモツを揃えることで「お通し」が発生し、高額売上（スコア）を叩き出せます。
🕹 遊び方

    移動: 左右の矢印キー、または画面下の「◀」「▶」ボタン。

    攻撃: 自動で連射されます。

    コンボシステム: 敵（文字）を倒すと「伝票」に文字が溜まります。

        シロ レバ ガツ ナンコツ などの単語が揃うと、売上ボーナス！

        ビール が揃うと、画面が黄金色に輝くラッシュモードが発動。

    ボス: 50機のモツを捌くと、巨大な 「特大レバー」 が出現します。

⚙️ 攻略のコツ

    時間経過による加速: 営業時間が長引くほど（生き残るほど）、モツの動きが速くなります。

    品切れなし: モツを倒しても次々に補充されます。一瞬の油断が命取りです。

    梅割り自機: 下町の魂がこもった琥珀色の自機を信じてください。

🛠 動作環境

    モダンなWebブラウザ（Chrome, Safari, Edgeなど）

    PC / スマートフォン両対応

🚀 設置方法

    このリポジトリをクローンまたはダウンロードします。

    自機の画像として umewari.png を同じフォルダに配置してください。

    index.html をブラウザで開くだけで開店できます。
